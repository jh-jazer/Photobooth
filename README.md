# PhotoBooth Pro

A modern, premium photobooth application built with Vite, React, and Tailwind CSS.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open code reference in your browser (usually http://localhost:5173).

## Features

-   **Webcam Support**: Real-time camera feed with mirroring.
-   **Countdown**: 3-second timer for perfect poses.
-   **Filters**: Real-time CSS filters (B&W, Sepia, Noir).
-   **Photo Strip**: Automatically generates a 4-photo strip layout.
-   **Download**: One-click download of the generated strip.
-   **Premium UI**: Smooth animations, glassmorphism, and responsive design.
